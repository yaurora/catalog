Pwndrop is a self-deployable file hosting service for sending out red teaming payloads or securely sharing your private files over HTTP and WebDAV.

This App is supplied by TrueCharts, for more information visit the manual: [https://truecharts.org/docs/charts/stable/pwndrop](https://truecharts.org/docs/charts/stable/pwndrop)
